# Linux Mint 22.x (Ubuntu Noble) / Debian — APT CLI packages aligned with ../packages.txt
# (minus node/npm/uv/chezmoi/starship/obsidian).
#
# Node: prefer nvm (~/.zshrc); system node is optional.
# Obsidian: Flatpak (see run_once_install_flatpaks.sh) or upstream; Arch uses pacman.
#
# One-shot (matches bootstrap-cli.sh APT set):
#   sudo apt-get update
#   sudo apt-get install -y \
#     git curl wget zsh tmux kitty bat ripgrep fd-find eza lnav zoxide btop gh
#
# Debian/Ubuntu naming notes (handled by ~/.config/dotfiles/shell-extras.sh):
#   - bat → binary often `batcat`; aliases map `cat` when appropriate.
#   - fd-find → provides `fdfind`; aliases map `fd` when `fd` is absent.
#
# Not in default APT on many releases — install separately:
#   chezmoi:   curl -fsSL https://chezmoi.io/get | bash -s -- -b ~/.local/bin
#   uv:        curl -LsSf https://astral.sh/uv/install.sh | sh
#   starship:  curl -fsSL https://starship.rs/install.sh | sh -s -- -y -b ~/.local/bin   # bash only; zsh uses Powerlevel10k
#
# Full automation: bash linux/bootstrap-cli.sh
#
